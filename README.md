City Pulse

First complete the setup-

  Navigate to the folder City first
    -cd City

  Install dependencies for backend
    -npm install

  Install dependencies for frontend
    -cd client,
       -npm install

To run this-

  In first terminal(Start the backend)
    -cd City,
    -node app.js || 
   In second terminak(Start the frontend)
    -cd City,
    -cd client,
    -npm run dev,
